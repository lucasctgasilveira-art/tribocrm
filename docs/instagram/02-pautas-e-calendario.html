02 Estruturas-padrão de copy

Frameworks reutilizáveis. Formato → estrutura → regras. Vai virar template HTML no Bloco 3.

Carrossel-MÉTODO (8-10 slides) · formato principal

Salvamento como métrica primária
SLIDE 1 · CAPA → Hook + promessa específica + indicação de "arrasta" SLIDE 2 · DOR → Por que isso importa (reconhecimento) SLIDE 3 · ERRO COMUM → O que quase todo mundo faz errado SLIDE 4-7 · MÉTODO → 1 passo por slide (com micro-exemplo) SLIDE 8 · TEMPLATE → Mini-checklist ou template aplicável SLIDE 9 · ANCORAGEM → Síntese, princípio ou número público (não case de cliente — Prova suspenso 28/04) SLIDE 10 · CTA → "Salva pra rodar com seu time" + handle
  • Slide 1 e 10 carregam o post inteiro — caprichar
  • Cada slide ≤ 35 palavras
  • 1 ideia por slide; sem subtópicos
  • Contraste tipográfico: frase-chave grande, contexto pequeno

Carrossel-PROVA (6-8 slides) 🛑 SUSPENSO

Estrutura preservada — não usar até cliente real com permissão escrita
SLIDE 1 · CAPA → Nome do cliente + segmento + transformação SLIDE 2 · ESTADO ANTES → Print/dado/cena vívida do "antes" SLIDE 3 · A VIRADA → Qual decisão tomou (ato concreto) SLIDE 4-5 · O QUE MUDOU → Print do CRM, dashboard, número SLIDE 6 · FALA DO DONO → Citação direta com aspas SLIDE 7-8 · LIÇÃO + CTA → "Quer ver isso rodando? Link na bio"

Reel-DOR (15-30s · 3 atos)

Alcance via reconhecimento
0-3s → GANCHO DE RECONHECIMENTO ("Sabe quando...") 3-15s → DESCRIÇÃO VÍVIDA da cena (real, específica) 15-25s → PUNCH LINE (dar nome ao problema) 25-30s → CTA SUAVE ("Comenta se já viveu")
  • Frase-impacto nos primeiros 3 segundos (sem isso, perdeu)
  • Cena específica vence comentário abstrato
  • Punch line dá NOME — vira termo que cliente repete

Reel-MÉTODO (30-60s)

Educação em formato curto
0-3s → HOOK + PROMESSA 3-15s → O ERRO 15-45s → MÉTODO EM 3 PASSOS (10s cada) 45-60s → SÍNTESE DO MÉTODO + CTA "salva pra usar" (Prova suspenso · não usar case)

Reel-BASTIDOR (30-90s)

Pessoa Comum 70% · sem performance
0-5s → CONTEXTO ("Essa semana a gente teve uma decisão...") 5-60s → A HISTÓRIA (cena real) 60-80s → O APRENDIZADO 80-90s → CONVITE GENUÍNO ("Conta o que você acha")

Feed-MANIFESTO (1080×1350)

Manifesto ressoa, não converte direto
  • Frase única (5-12 palavras)
  • DM Sans 800, branco com palavra-chave em laranja
  • Símbolo D no canto inferior direito
  • Assinatura "TriboCRM" embaixo
  • Sem CTA

Feed-CITAÇÃO (1080×1350)

  • Citação 1-2 frases (cliente, especialista, frase-chave)
  • Atribuição (nome + papel + segmento)
  • Símbolo D + assinatura

Story-ENQUETE / BASTIDOR / RECAP

  • Enquete: pergunta provocativa + binária A/B + resposta no story seguinte
  • Bastidor: foto/vídeo crua + texto curto + convite a DM
  • Recap: final de semana, 3-5 stories com prints da semana + sticker pra cada
